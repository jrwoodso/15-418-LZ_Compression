# Parallel LZW Compression Suite

**
This repository contains a collection of Lempel-Ziv-Welch (LZW) compression algorithms, ranging from a baseline sequential approach to highly optimized parallel implementations. The project focuses on workload balancing and speedup optimization across high core counts and GPU architectures.

**Please note that the makefile was generated with LLMs, as well as the build instructions and usage examples below.**

## Implementations

* **Sequential (`lzw`)**: A baseline C++17 implementation utilizing memory mapping (`mmap`) for efficient zero-copy file I/O.
* **OpenMP Block Parallel (`lzw_parallel`)**: A multi-threaded compressor that dynamically partitions single files into independent blocks, compressing them concurrently using OpenMP.
* **OpenMP File/Directory Parallel (`lzw_file_parallel`)**: A task-based parallel tool designed for bulk processing. It recursively traverses directories, dispatching whole files to a thread pool and sub-dividing larger files into parallel block tasks.
* **CUDA GPU Compressor (`lzw_gpu_compress`)**: A highly optimized GPU implementation utilizing CUDA. Designed for structured data grids (e.g., 4096×3072 images), it transposes memory and processes independent strips in parallel.
* **GPU Decompressor (`lzw_gpu_decompress`)**: A specialized CPU-based decompressor built to decode the strict flush-boundary format generated by the CUDA compressor.

## Build Instructions

The project uses a standard `Makefile`. You will need `g++` (with OpenMP and C++17 support) and `nvcc` (CUDA Toolkit).

To build all executables, run:
```bash
make
```

To clean up compiled binaries:
```bash
make clean
```

## Usage Examples

### 1. Sequential & Block Parallel LZW
Both the sequential and single-file parallel versions use the same CLI structure. You can optionally specify the number of threads for the parallel version.

```bash
# Compress a file
./lzw -c input.txt compressed.lzw
./lzw_parallel -c input.txt compressed.lzw [num_threads]

# Decompress a file
./lzw -d compressed.lzw output.txt
./lzw_parallel -d compressed.lzw output.txt [num_threads]
```

### 2. Directory Parallel LZW
Compress or decompress entire folders recursively. 

```bash
# Compress a directory
./lzw_file_parallel -c ./my_data ./compressed_output [num_threads]

# Decompress a directory
./lzw_file_parallel -d ./compressed_output ./restored_data [num_threads]
```

### 3. CUDA GPU Image Compression
The GPU compressor is hardcoded to expect raw binary input matching a 4096×3072 grid. 

```bash
# Compress raw image data on the GPU
./lzw_gpu_compress input_4096x3072.raw compressed_image.bin

# Decompress back to raw format on the CPU
./lzw_gpu_decompress compressed_image.bin output_4096x3072.raw
```
